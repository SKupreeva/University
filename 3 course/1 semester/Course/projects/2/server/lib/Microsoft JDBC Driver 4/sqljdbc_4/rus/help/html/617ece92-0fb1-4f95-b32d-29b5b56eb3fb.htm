Метод setAsciiStream (SQLServerNClob)

Извлекает значение потока, которое используется для записи символов ASCII в значение NCLOB, представляемое этим объектом java.sql.NClob, начиная с начального положения.

Синтаксис

public java.io.OutputStream setAsciiStream(long pos)

Параметры

pos

Позиция, с которой начинается запись в объект NCLOB (отсчитывается от 1).

Возвращаемое значение

Объект OutputStream, представляющий поток, в который можно записывать символы в кодировке ASCII.

Исключения

SQLServerException

Замечания

Этот метод setAsciiStream указывается с помощью метода setAsciiStream в интерфейсе java.sql.NClob.
