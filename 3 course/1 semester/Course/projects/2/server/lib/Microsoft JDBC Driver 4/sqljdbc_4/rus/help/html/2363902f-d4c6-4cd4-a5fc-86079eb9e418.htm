Методы

Имя

Описание

addBatch

Добавляет набор параметров к пакету команд для этого объекта Statement.

cancel

(Наследуется от SQLServerStatement.) Отменяет инструкцию SQL, выполняемую в настоящее время объектом Statement.

clearBatch

Обнуляет текущий список команд SQL для данного объекта SQLServerStatement.

clearParameters

Немедленно удаляет текущие значения параметров.

clearWarnings

(Наследуется от SQLServerStatement.) Удаляет все предупреждения, включенные в отчет для этого объекта Statement.

close

Немедленно высвобождает ресурсы базы данных и JDBC этого объекта Statement вместо ожидания их автоматического освобождения.

execute

Выполняет инструкцию SQL в этом объекте Statement, который может принадлежать к любому типу инструкции SQL.

executeBatch

Передает пакет команд базе данных для выполнения. В случае успешного выполнения всех команд возвращает массив из количества операций обновления, выполненных той или иной командой.

executeQuery

Выполняет запрос SQL в этом объекте Statement и возвращает объект SQLServerResultSet, созданный этим запросом.

executeUpdate

Выполняет инструкцию SQL в этом объекте Statement. Это должна быть инструкция SQL INSERT, UPDATE, MERGE или DELETE либо инструкция SQL, не возвращающая значения, например инструкция DDL.

getConnection

(Наследуется от SQLServerStatement.) Извлекает объект SQLServerConnection, на основе которого был создан этот объект Statement.

getFetchDirection

(Наследуется от SQLServerStatement.) Извлекает направление выборки строк из таблиц базы данных, заданное по умолчанию для результирующих наборов, созданных на основе этого объекта Statement.

getFetchSize

(Наследуется от SQLServerStatement.) Извлекает количество строк результирующего набора, заданное как размер выборки по умолчанию для объектов результирующих наборов, созданных на основе этого объекта Statement.

getGeneratedKeys

(Наследуется от SQLServerStatement.) Извлекает все автоматически сформированные ключи, созданные в результате выполнения этого объекта Statement.

getMaxFieldSize

(Наследуется от SQLServerStatement.) Извлекает максимальное число байтов, которое может возвращаться для значений символьных и двоичных столбцов в объекте SQLServerResultSet, созданном этим объектом Statement.

getMaxRows

(Наследуется от SQLServerStatement.) Извлекает максимальное количество строк, которое может содержаться в объекте SQLServerResultSet, созданном этим объектом Statement.

getMetaData

Извлекает объект Класс SQLServerResultSetMetaData, содержащий информацию о столбцах объекта SQLServerResultSet, который будет возвращен при выполнении данного объекта Statement.

getMoreResults

(Наследуется от SQLServerStatement.) Переходит к следующему результату этого объекта Statement.

getParameterMetaData

Извлекает количество, типы и свойства параметров этого объекта Statement.

getResponseBuffering

(Наследуется от SQLServerStatement.) Извлекает режим буферизации ответов для этого объекта SQLServerStatement.

getQueryTimeout

(Наследуется от SQLServerStatement.) Извлекает время в секундах, в течение которого драйвер Драйвер Microsoft JDBC для SQL Server будет ждать выполнения данного Statement.

getResultSet

(Наследуется от SQLServerStatement.) Извлекает текущий результат как объект SQLServerResultSet.

getResultSetConcurrency

(Наследуется от SQLServerStatement.) Извлекает значение параллелизма результирующего набора объектов SQLServerResultSet, созданных этим объектом Statement.

getResultSetHoldability

(Наследуется от SQLServerStatement.) Возвращает признак удержания результирующего набора для объектов SQLServerResultSet, созданных этим объектом Statement.

getResultSetType

(Наследуется от SQLServerStatement.) Возвращает тип результирующего набора для объектов SQLServerResultSet, созданных этим объектом Statement.

getUpdateCount

(Наследуется от SQLServerStatement,) Возвращает текущий результат в виде счетчика обновлений.

getWarnings

(Наследуется от SQLServerStatement.) Извлекает первое предупреждение, указанное в отчетах для вызовов этого объекта Statement.

isClosed

(Наследуется от SQLServerStatement.) Указывает, был ли закрыт данный объект Statement.

isPoolable

(Наследуется от SQLServerStatement.) Возвращает значение, показывающее, можно ли добавить инструкцию в предоставляемый пользователем пул инструкций.

isWrapperFor

Указывает, является ли этот объект инструкции оболочкой указанного интерфейса.

setArray

Задает номер указанного параметра для заданного объекта Array.

setAsciiStream

Задает номер указанного параметра для заданного объекта InputStream.

setBigDecimal

Задает номер указанного параметра для заданного объекта BigDecimal.

setBinaryStream

Устанавливает для указанного параметра указанное значение входного потока.

setBlob

Задает указанный параметр для заданного объекта Blob.

setboolean

Задает для указанного параметра указанное значение типа Boolean.

setByte

Задает для указанного параметра указанное значение типа byte.

setBytes

Задает указанный параметр для указанного массива байтов.

setCharacterStream

Задает указанный параметр для заданного объекта Reader.

setClob

Задает указанный параметр для заданного объекта Clob.

setCursorName

(Наследуется от SQLServerStatement.) Для имени курсора SQL задается значение типа String, которое будет использоваться последующими методами выполнения.

setDate

Устанавливает для указанного параметра указанное значение даты.

setDateTimeOffset

Устанавливает указанный столбец в значение Класс DateTimeOffset.

setDouble

Задает для указанного параметра указанное значение типа double.

setEscapeProcessing

(Наследуется от SQLServerStatement.) Задается режим обработки escape-последовательностей.

setFetchDirection

(Наследуется от SQLServerStatement.) Определяет указание для драйвера JDBC относительно направления, в котором будут обрабатываться строки результирующего набора.

setFetchSize

(Наследуется от SQLServerStatement.) Определяет указание для драйвера JDBC относительно числа строк, которые должны быть извлечены из базы данных при необходимости в дополнительных строках.

setFloat

Задает для указанного параметра указанное значение типа float.

setInt

Задает для указанного параметра указанное значение типа int.

setLong

Задает для указанного параметра указанное значение типа long.

setMaxFieldSize

(Наследуется от SQLServerStatement.) Устанавливает ограничение на максимальное число байтов для столбца SQLServerResultSet, в котором хранятся символьные или двоичные значения.

setMaxRows

(Наследуется от SQLServerStatement.) Устанавливает равное заданному числу ограничение для максимального количества строк, которое может содержаться в объекте SQLServerResultSet.

setNCharacterStream

Задает указанный параметр для заданного объекта Reader.

setNClob

Задает указанный параметр для заданного объекта.

setNull

Устанавливает значение NULL для параметра, определяемого по заданному типу.

setNString

Задает указанный параметр для заданного объекта String.

setObject

Устанавливает значение указанного параметра с помощью заданного объекта.

setPoolable

(Наследуется от SQLServerStatement.) Запрашивает поддержку пулов или запрет пулов в инструкции. По умолчанию объект SQLServerPreparedStatement во время создания поддерживает организацию пула.

setQueryTimeout

(Наследуется от SQLServerStatement.) Задает количество секунд, в течение которых драйвер будет ожидать выполнения объекта Statement в течение указанного периода времени.

setRef

Задает указанный параметр для заданного объекта Ref.

setResponseBuffering

(Наследуется от SQLServerStatement.) Задает режим буферизации ответов для этого объекта SQLServerStatement — String full без учета регистра или adaptive.

setShort

Задает для указанного параметра указанное значение типа short.

setString

Задает для указанного параметра указанное значение типа String.

setSQLXML

Задает указанный параметр для заданного объекта SQLXML.

setTime

Устанавливает для указанного параметра указанное значение времени.

setTimestamp

Устанавливает для указанного параметра указанное значение отметки времени.

setUnicodeStream

Присваивает указанный номер параметра заданному входному потоку, который будет содержать указанное число байтов.

setURL

Устанавливает для указанного параметра указанное значение URL-адреса.

unwrap

Возвращает объект, реализующий указанный интерфейс для доступа к методам, относящимся к драйверу Драйвер Microsoft JDBC для SQL Server.
