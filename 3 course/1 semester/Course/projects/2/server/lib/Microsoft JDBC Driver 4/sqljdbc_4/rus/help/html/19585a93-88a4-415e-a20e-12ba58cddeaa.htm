Метод getResponseBuffering (SQLServerDataSource)

Возвращает режим буферизации ответов для этого объекта SQLServerDataSource.

Синтаксис

public java.lang.String getResponseBuffering()

Возвращаемое значение

Значение String, который содержит full в нижнем регистре или adaptive.

Замечания

Значение full указывает на чтение всех результатов с сервера во время выполнения.

Значение adaptive указывает на буферизацию необходимого минимума данных. Значение adaptive является режимом буферизации по умолчанию.

Дополнительные сведения об использовании режима буферизации ответов см. в разделе Использование адаптивной буферизации.
