Элементы SQLServerResultSetMetaData

В следующих таблицах перечислены элементы, предоставляемые классом SQLServerResultSetMetaData.

Конструкторы

None.

Поля

None.

Наследуемые поля

Имя

Описание

java.sql.ResultSetMetaData

columnNoNulls, columnNullable, columnNullableUnknown

Методы

Имя

Описание

getCatalogName

Возвращает имя каталога для таблицы, содержащей указанный столбец.

getColumnClassName

Возвращает полное имя Java-класса, экземпляры которого создаются при вызове метода getObject класса SQLServerResultSet для получения значения столбца.

getColumnCount

Возвращает число столбцов в результирующем наборе.

getColumnDisplaySize

Возвращает стандартную максимальную ширину заданного столбца в символах.

getColumnLabel

Возвращает название, предполагаемое для использования в распечатках, и отображает заданный столбец.

getColumnName

Возвращает имя указанного столбца.

getColumnType

Возвращает тип SQL указанного столбца.

getColumnTypeName

Возвращает имя определяемого базой данных типа для указанного столбца.

getPrecision

Возвращает число десятичных разрядов для указанного столбца.

getScale

Возвращает число цифр справа от десятичной запятой для указанного столбца.

getSchemaName

Возвращает имя схемы для таблицы, содержащей указанный столбец.

getTableName

Возвращает имя таблицы для указанного столбца.

isAutoIncrement

Указывает, производится ли автоматическая нумерация указанного столбца, что делает его доступным только для чтения.

isCaseSensitive

Указывает, учитывается ли регистр символов для столбца.

isCurrency

Указывает, содержит ли указанный столбец значения денежных сумм.

isDefinitelyWritable

Указывает, будет ли запись в указанный столбец наверняка успешной.

isNullable

Указывает допустимость значений NULL для указанного столбца.

isReadOnly

Указывает, является ли указанный столбец наверняка недоступным для чтения.

isSearchable

Указывает, может ли указанный столбец использоваться в предложении SQL WHERE.

isSigned

Указывает, являются ли значения в указанном столбце числами со знаком.

isSparseColumnSet

Указывает, является ли столбец из результирующего набора набором разреженных столбцов.

isWritable

Указывает, возможна ли успешная запись в указанный столбец.
