Метод getDriverMinorVersion (SQLServerDatabaseMetaData)

Возвращает дополнительный номер версии данного драйвера JDBC.

Синтаксис

public int getDriverMinorVersion()

Возвращаемое значение

Значение типа int, которое указывает дополнительный номер версии драйвера JDBC.

Исключения

SQLServerException

Замечания

Этот метод getDriverMinorVersion указывается методом getDriverMinorVersion в интерфейсе java.sql.DatabaseMetaData.
