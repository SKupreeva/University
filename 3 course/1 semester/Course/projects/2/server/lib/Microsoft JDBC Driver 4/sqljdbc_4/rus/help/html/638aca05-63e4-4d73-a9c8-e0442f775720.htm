Метод getParameterType (SQLServerParameterMetaData)

Возвращает тип SQL указанного параметра.

Синтаксис

public int getParameterType(int param)

Параметры

param

Значение типа int, указывающее индекс параметра.

Возвращаемое значение

Значение типа int, указывающее код типа JDBC, определенный в java.sql.Types.

Исключения

SQLServerException

Замечания

Этот метод getParameterType указывается в методе getParameterType в интерфейсе java.sql.ParameterMetaData.
