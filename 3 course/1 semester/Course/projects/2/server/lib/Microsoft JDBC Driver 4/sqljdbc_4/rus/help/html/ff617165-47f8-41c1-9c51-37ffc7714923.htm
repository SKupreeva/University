Метод getBytes (java.lang.String) (SQLServerResultSet)

Получает значение имени заданного столбца в текущей строке этого объекта SQLServerResultSet в виде массива типа byte на языке программирования Java.

Синтаксис

public byte[] getBytes(java.lang.String columnName)

Параметры

columnName

Значение String, содержащее имя столбца.

Возвращаемое значение

Массив значений типа byte.

Исключения

SQLServerException

Замечания

Этот метод getBytes указывается с помощью метода getBytes в интерфейсе java.sql.ResultSet.

Этот метод поддерживает получение всех столбцов путем прямого считывания байтов с сервера. Он возвращает массив байтов непосредственно с сервера в формате, который хранится на сервере.

В предыдущей версии Драйвер Microsoft JDBC для SQL Server метод SQLServerResultSet.getBytes мог преобразовывать значения из байтового массива в типы данных SQL Server date, time, datetime2 или datetimeoffset и обратно. Теперь при вызове этого метода с такими типами данных возникает исключение, указывающее, что такое преобразование не поддерживается.
