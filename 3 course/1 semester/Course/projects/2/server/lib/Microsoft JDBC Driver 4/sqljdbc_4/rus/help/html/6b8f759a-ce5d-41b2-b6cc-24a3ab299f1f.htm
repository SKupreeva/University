Метод updateClob (java.lang.String, java.io.Reader, long)

Обновляет заданный столбец с помощью указанного объекта Reader, имеющего указанную длину в символах.

Синтаксис

public void updateClob(java.lang.String columnLabel,
                        java.io.Reader reader,
                        long length)

Параметры

columnLabel

Значение типа String, содержащее метку столбца.

reader

Объект Reader.

length

Количество символов в данных параметра.

Исключения

SQLServerException

Замечания

Этот метод updateClob указывается с помощью метода updateClob в интерфейсе java.sql.ResultSet.
