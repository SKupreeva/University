Метод getDateTimeOffset (SQLServerCallableStatement)

Этот метод добавлен в версии 3.0 драйвера Microsoft SQL Server JDBC.

Получает значение заданного параметра в виде объекта Класс DateTimeOffset на языке программирования Java по заданному индексу параметра.

Значение параметра Класс DateTimeOffset можно задать с помощью метода SQLServerCallableStatement.setDateTimeOffset.
