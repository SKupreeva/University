Метод setNull (int, int, java.lang.String)

Устанавливает указанный параметр в значение NULL по заданному имени и типу параметра.

Синтаксис

public final void setNull(int paramIndex,
                          int sqlType,
                          java.lang.String typeName)

Параметры

paramIndex

Значение int, указывающий номер параметра.

sqlType

Код типа JDBC, определенный в java.sql.Types.

typeName

Значение типа String, которое указывает полное имя задаваемого параметра.

Исключения

SQLServerException

Замечания

Этот метод setNull указывается методом setNull в интерфейсе java.sql.PreparedStatement.
