Метод getNString (int)

Получает значение указанного параметра NCHAR, NVARCHAR или LONGNVARCHAR в виде объекта String на языке программирования Java.

Синтаксис

public final java.lang.String getNString(int parameterIndex)

Параметры

parameterIndex

Значение int, указывающее индекс параметра.

Возвращаемое значение

ОбъектString.

Исключения

SQLServerException

Замечания

Этот метод getNString указывается с помощью метода getNString в интерфейсе java.sql.CallableStatement.
