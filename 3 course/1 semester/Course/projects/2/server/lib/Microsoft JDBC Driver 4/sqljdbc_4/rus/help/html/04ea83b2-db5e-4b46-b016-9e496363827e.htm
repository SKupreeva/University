Метод setTime (SQLServerCallableStatement)

Устанавливает указанный параметр в заданное значение времени.

Начиная с версии 3.0 драйвера SQL Server JDBC, на работу этого метода влияет свойство соединения sendTimeAsDatetime (Задание свойств соединения) и SQLServerDataSource.setSendTimeAsDatetime.

Дополнительные сведения см. в разделе Настройка способа отправки значений java.sql.Time на сервер.
