Справочник интерфейса драйвера JDBC

Драйвер Драйвер Microsoft JDBC для SQL Server содержит API, который может использоваться в программном коде Java для установки соединений и взаимодействия с базой данных Microsoft SQL Server.

Основные сведения об использовании драйвера JDBC см. в разделе Общие сведения о драйвере JDBC.

Имя интерфейса

Описание

Интерфейс ISQLServerCallableStatement

Позволяет указать имя вызываемой хранимой процедуры с входными и выходными параметрами.

Интерфейс ISQLServerConnection

Представляет соединение JDBC с базой данных SQL Server.

Класс SQLServerDataSource Class

Представляет список свойств, относящихся к соединению с базой данных SQL Server с помощью объекта ISQLServerConnection.

ISQLServerPreparedStatement

Представляет базовую реализацию функциональности подготовленной инструкции JDBC.

ISQLServerResultSet

Представляет результирующий набор JDBC.

ISQLServerStatement

Представляет базовую реализацию функциональности инструкции JDBC.

Классы

Имя класса

Описание

DateTimeOffset

Представляет объект типа microsoft.sql.DateTimeOffset.

SQLServerBlob

Представляет большой двоичный объект (BLOB).

SQLServerCallableStatement

Реализует ISQLServerCallableStatement.

SQLServerClob

Представляет символьный большой двоичный объект (CLOB).

SQLServerConnection

Реализует ISQLServerConnectopn.

SQLServerConnectionPoolDataSource

Представляет физические подключение к базе данных для диспетчеров пулов соединений.

SQLServerDatabaseMetaData

Представляет метаданные для базы данных.

SQLServerDataSource

Представляет список свойств, относящихся к подключению к базе данных SQL Server с использованием объекта SQLServerConnection.

SQLServerDataSourceObjectFactory

Представляет фабрику объектов для материализации источников данных из интерфейса JNDI.

SQLServerDriver

Представляет драйвер JDBC. Этот класс включает методы для соединения с базой данных SQL Server и для получения данных о драйвере JDBC.

SQLServerException

Представляет неуспешное или неполное выполнение инструкции SQL.

Класс SQLServerNClob

Представляет символьный большой двоичный объект, используя национальный набор символов.

SQLServerParameterMetaData

Представляет метаданные для параметров подготовленных инструкций.

SQLServerPooledConnection

Представляет физическое подключение к базе данных в пуле соединений.

SQLServerPreparedStatement

Реализует ISQLServerPreparedStatement.

SQLServerResource

Представляет локализованный ресурс строки ошибки. Этот класс предназначен только для внутреннего использования.

SQLServerResultSet

Реализует ISQLServerResultSet.

SQLServerResultSetMetaData

Представляет метаданные столбцов, которые содержатся в результирующем наборе.

SQLServerSavepoint

Представляет контрольную точку, для которой может быть выполнен откат транзакции.

SQLServerStatement

Реализует ISQLServerStatement.

SQLServerXAConnection

Представляет подключения JDBC, которые могут участвовать в распределенных транзакциях (XA).

SQLServerXADataSource

Представляет фабрику для объектов SQLServerXAConnection, предназначенных для внутреннего использования.

SQLServerXAResource

Представляет XAResource для управления распределенными транзакциями XA.
