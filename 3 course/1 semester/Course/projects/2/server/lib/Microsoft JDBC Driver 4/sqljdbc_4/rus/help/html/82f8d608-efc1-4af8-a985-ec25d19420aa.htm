Метод registerOutParameter (SQLServerCallableStatement)

Регистрирует параметр OUT.

Список перегрузок

Имя

Описание

registerOutParameter (int, int)

Регистрирует параметр OUT с указанным порядковым номером для заданного типа JDBC.

registerOutParameter (int, int, int)

Регистрирует параметр OUT с указанным порядковым номером для заданного типа и масштаба JDBC.

registerOutParameter (int, int, java.lang.String)

Регистрирует параметр OUT с указанным порядковым номером для заданного типа и имени типа JDBC.

registerOutParameter (java.lang.String, int)

Регистрирует параметр OUT с указанным именем для заданного типа JDBC.

registerOutParameter (java.lang.String, int, int)

Регистрирует параметр OUT с указанным именем для заданного типа и масштаба JDBC.

registerOutParameter (java.lang.String, int, java.lang.String)

Регистрирует параметр OUT с указанным именем для заданного типа и имени типа JDBC.
