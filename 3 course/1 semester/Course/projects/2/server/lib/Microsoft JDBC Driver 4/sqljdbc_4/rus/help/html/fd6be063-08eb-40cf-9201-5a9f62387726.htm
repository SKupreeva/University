Метод setBinaryStream (int, java.io.InputStream, int)

Присваивает указанному параметру указанный входной поток, который будет содержать указанное число байтов.

Синтаксис

public final void setBinaryStream(int n,
                                  java.io.InputStream x,
                                  int length)

Параметры

n

Значение типа int, указывающее номер параметра.

x

Объект InputStream.

length

Значение типа int, определяющее число байтов.

Исключения

SQLServerException

Замечания

Этот метод setBinaryStream указывается методом setBinaryStream в интерфейсе java.sql.PreparedStatement.

Если длина потока отличается от указанной в параметре length, то драйвер JDBC вызовет исключение при обновлении или вставке строки.

Если длина потока неизвестна, то для параметра length может быть задано значение -1, после чего драйвер будет принимать потоки независимо от их длины. Для sqljdbc4.jar рекомендуется использовать метод JDBC 4.0 Метод setBinaryStream (int, java.io.InputStream), если приложение обновляет столбец с помощью потока, длина которого неизвестна.
