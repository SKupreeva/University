Члены SQLServerCallableStatement

В следующих таблицах перечислены доступные члены класса SQLServerCallableStatement.

Конструкторы

None.

Поля

Класс, из которого наследуется:

Методы

java.sql.Statement

CLOSE_ALL_RESULTS, CLOSE_CURRENT_RESULT, EXECUTE_FAILED, KEEP_CURRENT_RESULT, NO_GENERATED_KEYS, RETURN_GENERATED_KEYS, SUCCESS_NO_INFO

Наследуемые поля

Нет.

Методы

Имя

Описание

addBatch

(Наследуется от SQLServerPreparedStatement.) Добавляет набор параметров к пакету команд для этого объекта CallableStatement.

cancel

(Наследуется от SQLServerStatement.) Отменяет инструкцию SQL, выполняемую в настоящее время объектом CallableStatement.

clearBatch

(Наследуется от SQLServerPreparedStatement.) Обнуляет текущий список команд SQL для данного объекта CallableStatement.

clearParameters

(Наследуется от SQLServerPreparedStatement.) Немедленно удаляет текущие значения параметров.

clearWarnings

(Наследуется от SQLServerStatement.) Удаляет все предупреждения, включенные в отчет для этого объекта CallableStatement.

close

(Наследуется от SQLServerPreparedStatement.) Немедленно высвобождает ресурсы базы данных и JDBC этого объекта CallableStatement вместо ожидания их автоматического освобождения.

execute

(Наследуется от SQLServerPreparedStatement.) Выполняет инструкцию SQL в этом объекте CallableStatement, который может принадлежать к любому типу инструкции SQL.

executeBatch

(Наследуется от SQLServerPreparedStatement.) Передает пакет команд базе данных для выполнения. В случае успешного выполнения всех команд возвращает массив из количества операций обновления, выполненных той или иной командой.

executeQuery

(Наследуется от SQLServerPreparedStatement.) Выполняет запрос SQL в этом объекте CallableStatement и возвращает объект SQLServerResultSet, созданный этим запросом.

executeUpdate

(Наследуется от SQLServerPreparedStatement.) Выполняет инструкцию SQL в этом объекте CallableStatement. Это должна быть инструкция SQL INSERT, UPDATE, MERGE или DELETE либо инструкция SQL, не возвращающая значения, например инструкция DDL.

getConnection

(Наследуется от SQLServerStatement.) Извлекает объект SQLServerConnection, на основе которого был создан этот объект CallableStatement.

getDateTimeOffset

Получает значение указанного столбца в виде объекта Класс DateTimeOffset.

getFetchDirection

(Наследуется от SQLServerStatement.) Извлекает направление выборки строк из таблиц базы данных, заданное по умолчанию для результирующих наборов, созданных на основе этого объекта CallableStatement.

getFetchSize

(Наследуется от SQLServerStatement.) Извлекает количество строк результирующего набора, заданное как размер выборки по умолчанию для объектов результирующих наборов, созданных на основе этого объекта CallableStatement.

getGeneratedKeys

(Наследуется от SQLServerStatement.) Извлекает все автоматически сформированные ключи, созданные в результате выполнения этого объекта CallableStatement.

getMaxFieldSize

(Наследуется от SQLServerStatement.) Извлекает максимальное число байтов, которое может возвращаться для значений символьных и двоичных столбцов в объекте SQLServerResultSet, созданном этим объектом CallableStatement.

getMaxRows

(Наследуется от SQLServerStatement.) Извлекает максимальное количество строк, которое может содержаться в объекте SQLServerResultSet, созданном этим объектом CallableStatement.

getMetaData

(Наследуется от SQLServerPreparedStatement.) Возвращает объект Класс SQLServerResultSetMetaData, содержащий сведения о столбцах объекта SQLServerResultSet, который будет возвращен при запуске данного объекта CallableStatement.

getMoreResults

(Наследуется от SQLServerStatement.) Переходит к следующему результату этого объекта CallableStatement.

getParameterMetaData

(Наследуется от SQLServerPreparedStatement.) Извлекает количество, типы и свойства параметров этого объекта CallableStatement.

getArray

Возвращает значение указанного параметра в виде объекта Array.

Возвращает значение указанного параметра в виде потока символов ASCII.
