Метод setDate (int, java.sql.Date, java.util.Calendar)

Устанавливает указанный параметр в заданные значения даты и календаря.

Синтаксис

public final void setDate(int n,
                          java.sql.Date x,
                          java.util.Calendar cal)

Параметры

n

Значение int, указывающий номер параметра.

x

Объект Date.

cal

Объект Calendar.

Исключения

SQLServerException

Замечания

Этот метод setDate указывается методом setDate в интерфейсе java.sql.PreparedStatement.
