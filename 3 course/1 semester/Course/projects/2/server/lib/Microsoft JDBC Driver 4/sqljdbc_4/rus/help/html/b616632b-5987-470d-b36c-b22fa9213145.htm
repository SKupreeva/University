Класс SQLServerDataSourceObjectFactory

Представляет фабрику объектов для материализации источников данных из интерфейса JNDI.

Пакет: com.microsoft.sqlserver.jdbc

Расширяет: java.lang.Object

Реализует: javax.naming.spi.ObjectFactory

Синтаксис

public class SQLServerDataSourceObjectFactory

Замечания

Этот метод наследуется всеми классами источников данных. В рамках поддержки интерфейса Referenceable Драйвер Microsoft JDBC для SQL Server предоставляет доступ к этому классу, реализующему интерфейс ObjectFactory. Серверы приложений Java вызовут getReference для класса источников данных, и при этом будет создан объект Reference, использующий имя класса внутри себя в качестве своей фабрики класса.

Если серверу Java Application требуется отменить ссылку на объект Reference, он создает экземпляр объекта SQLServerDataSourceObjectFactory и вызывает метод getObjectInstance, передавая объект Reference для извлечения экземпляра источника данных.
