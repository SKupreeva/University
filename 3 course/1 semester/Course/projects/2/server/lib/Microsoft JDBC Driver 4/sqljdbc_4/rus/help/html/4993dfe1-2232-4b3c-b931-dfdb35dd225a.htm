Метод updateObject (int, java.lang.Object)

Обновляет указанный столбец значением типа Object по заданному индексу столбца.

Синтаксис

public void updateObject(int index,
                         java.lang.Object obj)

Параметры

index

Значение int , указывающее индекс столбца.

obj

Значение Object .

Исключения

SQLServerException

Замечания

Этот метод updateObject указывается с помощью метода updateObject в интерфейсе java.sql.ResultSet.
