Образец приложений драйвера JDBC

Образцы приложений Драйвер Microsoft JDBC для SQL Server демонстрируют различные функции драйвера JDBC. Дополнительно они показывают хороший стиль программирования, которого можно придерживаться при использовании драйвера JDBC с базой данных SQL Server.

Все образцы приложений содержатся в файлах кода *.java. Их можно откомпилировать и запустить на локальном компьютере. Они расположены в различных вложенных папках в следующих расположениях:

<installation directory>\sqljdbc_<version>\<language>\help\samples

В подразделах данного раздела описан порядок настройки и выполнения образцов приложений, приводится обсуждение результатов работы образцов приложений.
