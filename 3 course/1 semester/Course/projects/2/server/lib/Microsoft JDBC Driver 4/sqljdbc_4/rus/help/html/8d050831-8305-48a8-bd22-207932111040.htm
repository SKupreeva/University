Метод setClientInfo (java.lang.String, java.lang.String)

Задает значение указанного свойства сведений о клиенте.

Синтаксис

public void setClientInfo (java.lang.String name,
                           java.lang.String value)

Параметры

name

Элемент String, содержащий имя свойства сведений о клиенте, которое необходимо задать.

value

Элемент String, содержащий значение, которое необходимо задать свойству сведений о клиенте.

Исключения

SQLServerException

Замечания

Этот метод setClientInfo определяется методом setClientInfo в интерфейсе java.sql.Connection.

Драйвер Microsoft JDBC для SQL Server не поддерживает свойства для доступа к сведениям о клиенте. В драйвере JDBC 2.0 этот метод формирует предупреждение для свойства. Приложения должны использовать метод getWarnings класса SQLServerConnection для извлечения предупреждения.
