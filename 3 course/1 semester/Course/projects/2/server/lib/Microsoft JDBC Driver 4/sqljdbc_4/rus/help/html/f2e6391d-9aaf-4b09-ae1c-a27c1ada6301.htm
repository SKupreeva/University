Метод getPooledConnection (java.lang.String, java.lang.String)

Пытается установить физическое соединение с базой данных, которое может использоваться как соединение из пула, по заданному имени пользователя и паролю.

Синтаксис

public javax.sql.PooledConnection getPooledConnection(java.lang.String user,
                                                      java.lang.String password)

Параметры

user

Значение типа String, содержащее имя пользователя.

passwword

Значение типа String, содержащее пароль.

Возвращаемое значение

Объект SQLServerPooledConnection.

Исключения

java.sql.SQLException

Замечания

Этот метод getPooledConnection указывается в методе getPooledConnection в интерфейсе javax.sql.ConnectionPoolDataSource.
