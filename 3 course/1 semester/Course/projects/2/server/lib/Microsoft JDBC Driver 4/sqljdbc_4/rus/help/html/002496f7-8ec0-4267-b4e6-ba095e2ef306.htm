Интерфейс ISQLServerResultSet

Представляет результирующий набор JDBC. Этот интерфейс добавлен в версии 3.0 драйвера SQL Server JDBC.

Пакет: com.microsoft.sqlserver.jdbc

Расширяет: java.sql.ResultSet

Синтаксис

public interface ISQLServerResultSet

Замечания

Этот интерфейс реализуется классом Класс SQLServerResultSet.

Этот интерфейс обеспечивает доступ к следующим методам, определяемым драйвером Драйвер Microsoft JDBC для SQL Server.
