Метод setTransactionTimeout (SQLServerXAResource)

Задает значение времени ожидания текущей транзакции для этого объекта SQLServerXAResource.

Синтаксис

public boolean setTransactionTimeout(int seconds)

Параметры

seconds

Значение int.

Возвращаемое значение

Значение true, если время ожидания задано успешно. В противном случае значение false.

Исключения

javax.transaction.xa.XAException

Замечания

Этот метод setTransactionTimeout указывается в методе setTransactionTimeout в интерфейсе javax.transaction.xa.XAResource.
