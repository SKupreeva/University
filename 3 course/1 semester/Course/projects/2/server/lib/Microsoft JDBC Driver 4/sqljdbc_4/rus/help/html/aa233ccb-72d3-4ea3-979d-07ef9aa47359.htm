Метод length (SQLServerBlob)

Возвращает число байтов в большом двоичном объекте.

Синтаксис

public long length()

Возвращаемое значение

Значение типа long, указывающее число байтов.

Исключения

SQLServerException

Замечания

Этот метод length указывается с помощью метода length в интерфейсе java.sql.Blob.
