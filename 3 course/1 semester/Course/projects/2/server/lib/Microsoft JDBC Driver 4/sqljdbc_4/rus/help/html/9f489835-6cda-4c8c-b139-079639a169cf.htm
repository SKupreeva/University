Метод setResponseBuffering (ISQLServerStatement)

Задает режим буферизации ответов для этого объекта SQLServerStatement без учета регистра String full или adaptive.

Синтаксис

public final void setResponseBuffering(java.lang.String value)

Параметры

value

Элемент String, содержащий режим буферизации ответов. Допустимый режим может быть одной из следующих строк без учета регистра: full или adaptive.

Исключения

SQLServerException

Замечания

Значение adaptive указывает на буферизацию необходимого минимума данных.

Значение full указывает, что с сервера во время выполнения считывается весь результат.

Значение adaptive используется по умолчанию в версиях 2.0 и 3.0 драйвера JDBC. Значение full использовалось по умолчанию до версии 2.0.

Метод setResponseBuffering позволяет отменить соединение responseBuffering свойства String для текущего объекта SQLServerStatement. Дополнительные сведения об использовании режима буферизации ответов см. в разделе Использование адаптивной буферизации.

Если приложение указывает недопустимое значение параметра для метода setResponseBuffering, возникает исключение SQLServerException.
