Метод getReference (SQLServerConnectionPoolDataSource)

Возвращает ссылку на этот объект SQLServerConnectionPoolDataSource.

Синтаксис

public javax.naming.Reference getReference()

Возвращаемое значение

Объект Reference.

Замечания

Этот метод getReference указывается в методе getReference в интерфейсе javax.naming.Referenceable. Он переопределяет метод getReference класса SQLServerDataSource.
