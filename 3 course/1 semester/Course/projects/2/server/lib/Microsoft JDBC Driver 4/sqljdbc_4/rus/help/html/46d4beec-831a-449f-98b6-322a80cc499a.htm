Метод position (java.lang.String, long) (SQLServerNClob)

Возвращает позицию символа, в которой указанная подстрока searchstr находится в значении NCLOB, представленном данным объектом NClob.

Синтаксис

public long position(java.lang.String searchstr,
              long start)

Параметры

searchstr

Подстрока для поиска.

start

Позиция, с которой начнется поиск; отсчитывается от 1.

Возвращаемое значение

Позиция, в которой находится подстрока, либо значение -1, если она не найдена. Позиция отсчитывается от 1.

Исключения

SQLServerException

Замечания

Этот метод position указывается с помощью метода position в интерфейсе java.sql.NClob.
