Конструктор SQLServerBlob (SQLServerConnection, byte[])

Инициализирует новый экземпляр класса SQLServerBlob, если имеется объект SQLServerConnection и массив byte.

Этот метод устарел в версии драйвера JDBC 2.0. Используйте вместо него метод createBlob класса SQLServerConnection.

Синтаксис

public SQLServerBlob(SQLServerConnection connection,
                     byte[] data)

Параметры

connection

Объект SQLServerConnection.

data

Массив byte.
