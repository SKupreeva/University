Члены SQLServerStatement

В следующих таблицах перечислены члены, предоставляемые классом SQLServerStatement.

Конструкторы

Нет.

Поля

Нет.

Наследуемые поля

Имя

Описание

java.sql.Statement

CLOSE_ALL_RESULTS, CLOSE_CURRENT_RESULT, EXECUTE_FAILED, KEEP_CURRENT_RESULT, NO_GENERATED_KEYS, RETURN_GENERATED_KEYS, SUCCESS_NO_INFO

Методы

Имя

Описание

addBatch

Добавляет заданную команду SQL в текущий список команд для данного объекта SQLServerStatement.

cancel

Отменяет инструкцию SQL, выполняемую в настоящее время объектом SQLServerStatement.

clearBatch

Обнуляет текущий список команд SQL для данного объекта SQLServerStatement.

clearWarnings

Очищает все предупреждения, выданные для данного объекта SQLServerStatement.

close

Немедленно освобождает для данного объекта SQLServerStatement базу данных и ресурсы JDBC, не дожидаясь их автоматического освобождения.

execute

Выполняет заданную инструкцию SQL, которая может возвращать несколько результатов.

executeBatch

Передает пакет команд базе данных для выполнения. В случае успешного выполнения всех команд возвращает массив из количества операций обновления, выполненных той или иной командой.

executeQuery

Выполняет заданную инструкцию SQL и возвращает единичный объект SQLServerResultSet.

executeUpdate

Выполняет заданную инструкцию SQL, которой может быть инструкция INSERT, UPDATE, MERGE или DELETE, либо инструкцию SQL, не возвращающую значения, например инструкцию SQL DDL.

getConnection

Возвращает объект SQLServerConnection, создавший данный объект SQLServerStatement.

getFetchDirection

Возвращает направление выборки строк из таблиц базы данных, заданное по умолчанию для результирующих наборов, созданных на основе этого объекта SQLServerStatement.

getFetchSize

Возвращает число строк результирующего набора, равное размеру выборки по умолчанию для объектов результирующих наборов, созданных на основе этого объекта SQLServerStatement.

getGeneratedKeys

Возвращает все автоматически сформированные ключи, созданные в результате выполнения этого объекта SQLServerStatement.

getMaxFieldSize

Извлекает максимальное число байтов для возвращаемых значений символьных и двоичных столбцов в объекте SQLServerResultSet, созданном объектом SQLServerStatement.

getMaxRows

Извлекает максимальное количество строк, которое может содержаться в объекте SQLServerResultSet, созданном объектом SQLServerStatement.

getMoreResults

Переходит к следующему результату этого объекта SQLServerStatement.

getQueryTimeout

Получает время в секундах, в течение которого драйвер Драйвер Microsoft JDBC для SQL Server будет ожидать выполнения этого объекта SQLServerStatement.

getResponseBuffering

Извлекает режим буферизации ответов для этого объекта SQLServerStatement.

getResultSet

Извлекает текущий результат как объект SQLServerResultSet.

getResultSetConcurrency

Возвращает параллелизм результирующего набора для объектов SQLServerResultSet, созданных этим объектом SQLServerStatement.

getResultSetHoldability

Возвращает возможность сохранения результирующих наборов для объектов SQLServerResultSet, созданных этим объектом SQLServerStatement.

getResultSetType

Возвращает тип результирующего набора для объектов SQLServerResultSet, созданных этим объектом SQLServerStatement.

getUpdateCount

Возвращает текущий результат в виде счетчика обновлений.

getWarnings

Возвращает первое предупреждение, указанное в отчетах для вызовов этого объекта SQLServerStatement.

isClosed

Указывает, был ли закрыт этот объект SQLServerStatement.

isPoolable

Возвращает значение, показывающее, можно ли добавить инструкцию в предоставляемый пользователем пул инструкций.

isWrapperFor

Указывает, является ли этот объект инструкции оболочкой указанного интерфейса.

setCursorName

Присваивает имени курсора SQL значение, равное заданной строке, которое будет использоваться при дальнейших вызовах метода execute.

setEscapeProcessing

Задается режим обработки escape-последовательностей.

setFetchDirection

Определяет указание для драйвера JDBC относительно направления, в котором будут обрабатываться строки результирующего набора.

setFetchSize

Определяет указание для драйвера JDBC относительно числа строк, которые должны быть извлечены из базы данных при необходимости в дополнительных строках.

setMaxFieldSize

Устанавливает ограничение на максимальное число байтов для столбца SQLServerResultSet, в котором хранятся символьные или двоичные значения.

setMaxRows

Устанавливает равное заданному числу ограничение для максимального количества строк, которое может содержаться в объекте SQLServerResultSet.

setPoolable

Запрашивает поддержку пулов или запрет пулов в инструкции.

setQueryTimeout

Задает время в секундах, в течение которого драйвер будет ожидать выполнения объекта SQLServerStatement.

setResponseBuffering

Задает режим буферизации ответов для этого объекта SQLServerStatement без учета регистра String full или adaptive.

unwrap

Возвращает объект, реализующий указанный интерфейс для доступа к методам, относящимся к драйверу Драйвер Microsoft JDBC для SQL Server.
