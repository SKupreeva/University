Метод executeUpdate (SQLServerStatement)

Выполняет заданную инструкцию SQL, которой может быть инструкция INSERT, UPDATE или DELETE; или инструкция SQL, не возвращающая значений, например инструкция SQL DDL. Начиная с версии 3.0 драйвера Microsoft SQL Server JDBC, метод executeUpdate возвращает правильное количество строк, обновленных в операции MERGE.
