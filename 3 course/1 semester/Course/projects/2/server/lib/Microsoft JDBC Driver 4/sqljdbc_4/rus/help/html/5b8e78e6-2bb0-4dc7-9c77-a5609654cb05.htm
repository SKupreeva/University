Метод getSuperTypes (SQLServerDatabaseMetaData)

Получает описание иерархий определяемых пользователем типов, определенных в заданной схеме этой базы данных.

В настоящее время этот метод не поддерживается в Драйвер Microsoft JDBC для SQL Server. При использовании этот метод всегда возвращает пустой результирующий набор.

Синтаксис

public java.sql.ResultSet getSuperTypes(java.lang.String catalog,
                                        java.lang.String schemaPattern,
                                        java.lang.String typeNamePattern)

Параметры

catalog

Значение String, содержащее имя каталога.

schemaPattern

Значение String, содержащее шаблон имени схемы.

tableNamePattern

Значение типа String, содержащее шаблон имени таблицы.

Возвращаемое значение

Объект SQLServerResultSet.

Исключения

SQLServerException

Замечания

Этот метод getSuperTypes указывается методом getSuperTypes в интерфейсе java.sql.DatabaseMetaData.
