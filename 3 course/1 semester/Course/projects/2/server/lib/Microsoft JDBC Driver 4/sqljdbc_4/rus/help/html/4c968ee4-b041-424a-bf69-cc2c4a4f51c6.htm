Метод setIntegratedSecurity (SQLServerDataSource)

Задает значение типа Boolean, определяющее, включено ли свойство integratedSecurity.

Синтаксис

public void setIntegratedSecurity(boolean enable)

Параметры

enable

true, если включено integratedSecurity. В противном случае значение false.

Замечания

Установите значение "true", чтобы SQL Server использовались учетные данные Windows для проверки подлинности пользователя или приложения. Если установлено значение true, то драйвер Драйвер Microsoft JDBC для SQL Server выполняет поиск учетных данных, которые уже были предоставлены при входе в сеть или на компьютер, в локальном кэше учетных данных компьютера. Если установлено значение "false", необходимо будет указать имя пользователя и пароль.

Данное свойство поддерживается только в операционных системах Windows Microsoft.

Дополнительные сведения об использовании встроенной проверки подлинности см. в разделе Формирование URL-адреса соединения.
