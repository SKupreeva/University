Метод setNCharacterStream (java.lang.String, java.io.Reader)

Присваивает параметру заданный объект Reader.

Синтаксис

public final void setNCharacterStream(java.lang.String parameterName,
                       java.io.Reader value)

Параметры

parameterName

Значение String, определяющее имя параметра.

value

Объект Reader.

Исключения

SQLServerException

Замечания

Этот метод setNCharacterStream указывается с помощью метода setNCharacterStream в интерфейсе java.sql.CallableStatement.

Этот метод должен использоваться для типов данных NCHAR, NVARCHAR, NTEXT и XML.
