Получение версии драйвера

Версию установленного драйвера Драйвер Microsoft JDBC для SQL Server можно получить следующими способами.

  • Вызовите методы getDriverMajorVersion, getDriverMinorVersion или getDriverVersion класса SQLServerDatabaseMetaData.

  • Версия указана в файле readme.txt, распространяемом вместе с продуктом.

Кроме того, имя драйвера JDBC возвращается методом getDriverName в классе SQLServerDatabaseMetaData. Он возвращает строку «Драйвер Microsoft JDBC 4.0 для SQL Server».

Далее представлен пример выходных данных методов класса SQLServerDatabaseMetaData:

getDriverName = Microsoft JDBC Driver 4.0 for SQL Server

getDriverMajorVersion = 4

getDriverMinorVersion = 0

getDriverVersion = 4.0.xxx.x

Здесь «xxx.x» представляет номер окончательной версии.
