Интерфейс ISQLServerConnection

Представляет соединение JDBC с базой данных Microsoft SQL Server. Этот интерфейс добавлен в версии 3.0 драйвера SQL Server JDBC.

Пакет: com.microsoft.sqlserver.jdbc

Расширяет: java.sql.Connection

Синтаксис

public interface ISQLServerConnection

Замечания

Этот интерфейс реализуется классом Класс SQLServerConnection.

Этот интерфейс обеспечивает доступ к следующему полю, определяемому драйвером Драйвер Microsoft JDBC для SQL Server.
