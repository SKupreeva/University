Метод getXAResource (SQLServerXAConnection)

Возвращает объект SQLServerXAResource, который будет использоваться диспетчером транзакций для управления участием этого объекта SQLServerXAConnection в распределенной транзакции.

Синтаксис

public javax.transaction.xa.XAResource getXAResource()

Возвращаемое значение

Объект XAResource.

Исключения

java.sql.SQLException

Замечания

Этот метод getXAResource указывается методом getXAResource в интерфейсе javax.sql.XAConnection.
