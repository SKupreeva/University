Метод getParameterMode (SQLServerParameterMetaData)

Возвращает режим указанного параметра.

Синтаксис

public int getParameterMode(int param)

Параметры

param

Значение типа int, указывающее индекс параметра.

Возвращаемое значение

Значение типа int, указывающее режим указанного параметра. Возможно одно из следующих значений:

ParameterMetaData.parameterModeIn

ParameterMetaData.parameterModeInOut

ParameterMetaData.parameterModeOut

ParameterMetaData.parameterModeUnknown

Исключения

SQLServerException

Замечания

Этот метод getParameterMode указывается в методе getParameterMode в интерфейсе java.sql.ParameterMetaData.
