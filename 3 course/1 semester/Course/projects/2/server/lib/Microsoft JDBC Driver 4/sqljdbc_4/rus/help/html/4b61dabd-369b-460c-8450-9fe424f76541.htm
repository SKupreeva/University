Методы

Имя

Описание

addConnectionEventListener

(Наследуется от SQLServerPooledConnection.) Регистрирует заданный прослушиватель событий для получения уведомлений о событии в этом объекте Connection.

close

(Наследуется от SQLServerPooledConnection.) Закрывает физическое соединение, представляемое этим объектом Connection.

getConnection

(Наследуется от SQLServerPooledConnection.) Создает дескриптор объекта для физического соединения, представляемого этим объектом Connection.

getXAResource

Возвращает объект SQLServerXAResource, который будет использоваться диспетчером транзакций для управления участием этого объекта SQLServerXAConnection в распределенной транзакции.

removeConnectionEventListener

(Наследуется от SQLServerPooledConnection.) Удаляет заданный прослушиватель событий.
