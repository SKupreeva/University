Метод getResponseBuffering (ISQLServerStatement)

Извлекает режим буферизации ответов для этого объекта SQLServerStatement.

Синтаксис

public final java.lang.String getResponseBuffering()

Возвращаемое значение

Значение String, который содержит full в нижнем регистре или adaptive.

Замечания

Значение adaptive указывает на буферизацию необходимого минимума данных.

Значение full указывает, что с сервера во время выполнения считывается весь результат.

Значение adaptive используется по умолчанию в версиях 2.0 и 3.0 драйвера JDBC. Значение full использовалось по умолчанию до версии 2.0.

Дополнительные сведения об использовании режима буферизации ответов см. в разделе Использование адаптивной буферизации.
