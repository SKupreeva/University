Метод supportsResultSetHoldability (SQLServerDatabaseMetaData)

Возвращает значение, определяющее, поддерживает ли эта база данных заданную возможность сохранения результирующих наборов.

Синтаксис

public boolean supportsResultSetHoldability(int holdability)

Параметры

holdability

Значение типа int, указывающее возможность сохранения результирующего набора. Возможны следующие значения:

ResultSet.HOLD_CURSORS_OVER_COMMIT

ResultSet.CLOSE_CURSORS_AT_COMMIT

Возвращаемое значение

true, если поддерживается. В противном случае значение false.

Исключения

SQLServerException

Замечания

Этот метод supportsResultSetHoldability указывается методом supportsResultSetHoldability в интерфейсе java.sql.DatabaseMetaData.
