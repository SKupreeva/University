Повышение производительности и надежности с помощью драйвера JDBC

Одна из сторон разработки приложений, характерная для всех приложений — постоянная необходимость улучшения производительности и надежности. Существует несколько способов достижения данных целей при использовании драйвера Драйвер Microsoft JDBC для SQL Server.

В подразделах данного раздела приводится описание различных способов улучшения производительности и надежности приложения при использовании драйвера JDBC.
