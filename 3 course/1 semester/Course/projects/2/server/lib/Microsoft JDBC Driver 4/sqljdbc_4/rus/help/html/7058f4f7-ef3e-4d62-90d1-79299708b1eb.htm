Конструктор SQLServerClob (SQLServerConnection, java.lang.String)

Инициализирует новый экземпляр класса SQLServerClob по заданному объекту SQLServerConnection и строке данных.

Этот метод устарел в версии драйвера JDBC 2.0. Вместо этого используйте метод createClob класса SQLServerConnection.

Синтаксис

public SQLServerClob(SQLServerConnection connection,
                     java.lang.String data)

Параметры

connection

Объект SQLServerConnection.

data

Данные CLOB.
