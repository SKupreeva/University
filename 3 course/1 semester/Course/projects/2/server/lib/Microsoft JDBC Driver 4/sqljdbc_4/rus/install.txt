Инструкции по установке
Драйвер Microsoft JDBC 4.0 для SQL Server


ВВЕДЕНИЕ
------------
Здесь представлены инструкции по распаковке файлов sqljdbc_<версия>_enu.exe
и sqljdbc_<версия>_enu.tar.gz.


WINDOWS
-------
1. Прочтите файл license.txt. Загрузка и использование драйвера 4.0 Microsoft JDBC для SQL Server означает принятие этих условий лицензии.

2. Загрузите файл sqljdbc_<версия>_enu.exe во временный каталог.

3. Запустите файл sqljdbc_<версия>_enu.exe.

4. Введите каталог установки, когда будет предложено. Рекомендуется распаковывать
   этот ZIP-файл в каталог по умолчанию в %ProgramFiles%:

	Драйвер Microsoft JDBC 4.0 для SQL Server

5. После распаковки пакета откройте справочную систему JDBC, открыв файл справки 
   %InstallationDirectory%\Microsoft JDBC Driver 4.0 for SQL Server\sqljdbc_<version>\<language>\
   help\default.htm. Справочная система откроется в веб-браузере.


UNIX
----
1. Прочтите файл license.txt. Загрузка и использование драйвера 4.0 Microsoft JDBC для SQL Server означает принятие этих условий лицензии.

2. Загрузите файл sqljdbc_<версия>_enu.tar.gz во временный каталог.

3. Чтобы распаковать сжатый TAR-файл, перейдите в каталог, в который нужно распаковать драйвер,
   и введите следующую команду:

	gzip -d sqljdbc_<версия>_enu.tar.gz

4. Чтобы распаковать TAR-файл, переместите его в каталог, в который нужно установить драйвер, 
   и введите следующую команду:

	tar -xf sqljdbc_<версия>_enu.tar

5. После распаковки пакета откройте справочную систему JDBC, открыв файл справки 
   %InstallationDirectory%/Microsoft JDBC Driver 4.0 for SQL Server/sqljdbc_<version>/<language>/
   help/default.htm. Справочная система откроется в веб-браузере по 
   умолчанию.
