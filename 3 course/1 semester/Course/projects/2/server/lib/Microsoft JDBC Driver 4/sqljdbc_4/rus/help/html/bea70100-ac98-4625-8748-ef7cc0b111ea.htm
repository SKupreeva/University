Метод setURL (SQLServerDataSource)

Задает URL-адрес, используемый для соединения с источником данных.

Синтаксис

public void setURL(java.lang.String url)

Параметры

url

Значение String, содержащее URL-адрес.

Замечания

По соображениям безопасности не следует включать пароль в URL-адрес, передаваемый в метод setURL. Так происходит из-за того, что на серверах приложений Java сторонних разработчиков в пользовательском интерфейсе настройки для источника данных часто выводится значение, заданное для свойства URL-адреса. Вместо этого используйте для задания значения пароля метод setPassword. На серверах приложений Java не отображается пароль, заданный в источнике данных в пользовательском интерфейсе настройки.

Если метод setURL не вызван перед методом getURL, то getURL возвращает значение по умолчанию — «jdbc:sqlserver://».
