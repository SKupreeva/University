Соединение с SQL Server с помощью драйвера JDBC

Одной из самых основных операций, выполняемых с помощью драйвера Драйвер Microsoft JDBC для SQL Server, является установление соединения с базой данных SQL Server. Взаимодействие с базой данных происходит через объект SQLServerConnection, и в силу исключительно простой архитектуры драйвера JDBC практически все полезные операции выполняются через объект SQLServerConnection.

Если SQL Server прослушивает только порт IPv6, установите системное свойство java.net.preferIPv6Addresses, чтобы убедиться, что IPv6 используется для подключения к SQL Server вместо IPv4.

System.setProperty("java.net.preferIPv6Addresses", "true");
