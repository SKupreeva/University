Метод compareTo (DateTimeOffset)

Сравнивает данный объект DateTimeOffset с другим объектом DateTimeOffset по критерию совпадения времени GMT.

Синтаксис

public int compareTo(DateTimeOffset other)

Параметры

Значение типа DateTimeOffset, которое нужно сравнить с текущим экземпляром.

Возвращаемое значение

В следующей таблице описывается значение, возвращаемое данным методом.

Возвращаемое значение

Описание

0

Оба объекта DateTimeOffset представляют одну и ту же точку во времени.

Отрицательное число

Данный объект DateTimeOffset представляет точку во времени, предшествующую other.

Положительное число

Данный объект DateTimeOffset представляет точку во времени, расположенную после other.

Замечания

Если два объекта DateTimeOffset имеют одно и то же время GMT, то дополнительное упорядочение объектов по смещению не производится.
