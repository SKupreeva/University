Метод getConnection (java.lang.String, java.lang.String)

Пытается установить соединение с источником данных, представляемым этим объектом SQLServerDataSource, с использованием заданного имени пользователя и пароля.

Синтаксис

public java.sql.Connection getConnection(java.lang.String username,
                                         java.lang.String password)

Параметры

username

Значение типа String, содержащее имя пользователя.

password

Значение типа String, содержащее пароль.

Возвращаемое значение

Объект SQLServerConnection.

Исключения

java.sql.SQLException

Замечания

Этот метод getConnection указывается методом getConnection в интерфейсе javax.sql.DataSource.

Вызов метода getConnection с именем пользователя или паролем, отличным от NULL, приведет к замене свойств имени пользователя и пароля, заданных в классе SQLServerDataSource при инициализации объекта SQLServerConnection. Например, если вызывающий объект вызвал методы setUser и setPassword в источнике данных, а затем вызывает метод getConnection и передает имя пользователя, отличное от NULL, или пароль, отличный от NULL, то имя пользователя и пароль, заданные методами setUser и setPassword, будут заменены именем пользователя и паролем, переданными в getConnection.

Имя пользователя и пароль, которые заданы в URL-адресе путем вызова метода setURL, не будут изменяться в этом случае.
