Метод setNClob (java.lang.String, java.sql.NClob)

Присваивает параметру заданный объект NClob.

Синтаксис

public final void setNClob(java.lang.String parameterName,
              java.sql.NClob value)

Параметры

parameterName

Значение String, содержащее имя параметра.

value

Объект NClob.

Исключения

SQLServerException

Замечания

Этот метод должен использоваться для типов данных параметров NCHAR, NVARCHAR, NTEXT и XML.

Этот метод setNClob указывается с помощью метода setNClob в интерфейсе java.sql.CallableStatement.
