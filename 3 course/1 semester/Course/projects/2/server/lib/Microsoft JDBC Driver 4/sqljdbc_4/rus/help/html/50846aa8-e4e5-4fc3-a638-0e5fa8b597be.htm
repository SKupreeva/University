Метод isWritable (SQLServerResultSetMetaData)

Указывает, возможна ли успешная запись в указанный столбец.

Синтаксис

public boolean isWritable(int column)

Параметры

column

Значение int, указывающее индекс столбца.

Возвращаемое значение

Значение true, если запись столбца завершится успешно. В противном случае значение false.

Исключения

SQLServerException

Замечания

Этот метод isWritable указывается с помощью метода isWritable в интерфейсе java.sql.ResultSetMetaData.
