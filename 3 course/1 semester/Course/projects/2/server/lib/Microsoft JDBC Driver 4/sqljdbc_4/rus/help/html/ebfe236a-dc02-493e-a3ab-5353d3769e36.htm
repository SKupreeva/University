Метод getPrimaryKeys (SQLServerDatabaseMetaData)

Возвращает описание столбцов первичного ключа заданной таблицы.

Синтаксис

public java.sql.ResultSet getPrimaryKeys(java.lang.String cat,
                                         java.lang.String schema,
                                         java.lang.String table)

Параметры

cat

Значение String, содержащее имя каталога.

schema

Значение типа String, содержащее имя схемы.

table

Значение типа String, содержащее имя таблицы.

Возвращаемое значение

Объект SQLServerResultSet.

Исключения

SQLServerException

Замечания

Этот метод getPrimaryKeys указывается методом getPrimaryKeys в интерфейсе java.sql.DatabaseMetaData.

Результирующий набор, возвращаемый методом getPrimaryKeys, включает следующие данные.

Имя

Тип

Описание

TABLE_CAT

String

Имя базы данных, в которой расположена указанная таблица.

TABLE_SCHEM

String

Схема для таблицы.

TABLE_NAME

String

Имя таблицы.

COLUMN_NAME

String

Имя столбца.

KEY_SEQ

short

Порядковый номер столбца в первичном ключе из нескольких столбцов.

PK_NAME

String

Имя первичного ключа.

Дополнительные сведения о данных, возвращаемых методом getPrimaryKeys, см. в разделе «sp_pkeys (Transact-SQL)» электронной документации по SQL Server.

Пример

Следующий пример показывает использование метода getPrimaryKeys для возврата сведений о первичных ключах таблицы Person.Contact в образце базы данных SQL Server 2005AdventureWorks.

public static void executeGetPrimaryKeys(Connection con) {
   try {
      DatabaseMetaData dbmd = con.getMetaData();
      ResultSet rs = dbmd.getPrimaryKeys("AdventureWorks", "Person", "Contact");
      ResultSetMetaData rsmd = rs.getMetaData();

      // Display the result set data.
      int cols = rsmd.getColumnCount();
      while(rs.next()) {
         for (int i = 1; i <= cols; i++) {
            System.out.println(rs.getString(i));
         }
      }
      rs.close();
   } 

   catch (Exception e) {
      e.printStackTrace();
   }
}
