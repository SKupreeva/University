Метод getObjectInstance (SQLServerDataSourceObjectFactory)

Получает экземпляр указанного объекта источника данных.

Синтаксис

public java.lang.Object getObjectInstance(java.lang.Object ref,
                                          javax.naming.Name name,
                                          javax.naming.Context c,
                                          java.util.Hashtable h)

Параметры

ref

Значение Object.

name

Имя объекта.

c

Контекст, связанный с указанным именем.

h

Среда, используемая при создании объекта.

Возвращаемое значение

Значение Object.

Исключения

java.sql.SQLException

Замечания

Этот метод getObjectInstance указывается в методе getObjectInstance в интерфейсе javax.naming.spi.ObectFactory.
