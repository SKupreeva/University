Метод getFloat (java.lang.String) (ISQLServerResultSet)

Получает значение имени заданного столбца в текущей строке этого объекта SQLServerResultSet в виде значения типа float на языке программирования Java.

Синтаксис

public float getFloat(java.lang.String columnName)

Параметры

columnName

Значение String, содержащее имя столбца.

Возвращаемое значение

Значение типа float.

Исключения

SQLServerException

Замечания

Этот метод getFloat указывается с помощью метода getFloat в интерфейсе java.sql.ResultSet.

Этот метод возвращает все числовые типы с точностью float Java.
