Метод setLoginTimeout (SQLServerDataSource)

Задает время ожидания в секундах для данного объекта SQLServerDataSource при попытке установить соединение.

Синтаксис

public void setLoginTimeout(int loginTimeout)

Параметры

loginTimeout

Значение int, которое определяет число секунд ожидания. Нулевое значение означает, что время ожидания равно системному времени ожидания по умолчанию, т.е. 15 секундам.

Замечания

Этот метод setLoginTimeout указывается методом setLoginTimeout в интерфейсе javax.sql.DataSource.
