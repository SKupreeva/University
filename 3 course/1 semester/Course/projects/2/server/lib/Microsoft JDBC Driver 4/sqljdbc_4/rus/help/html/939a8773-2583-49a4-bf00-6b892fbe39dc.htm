Общие сведения о драйвере JDBC

Драйвер Драйвер Microsoft JDBC для SQL Server совместим со спецификацией Java Database Connectivity 4.0 и обеспечивает стабильный доступ к данным в базах данных Microsoft SQL Server. Драйвер JDBC имеет доступ ко многим функциям SQL Server 2005, включая зеркальное отображение базы данных, типы данных xml, определяемые пользователем типы данных и типы данных большого объема. Драйвер поддерживает изоляцию моментальных снимков для транзакций. Помимо этого, драйвер JDBC поддерживает встроенную проверку подлинности.

В подразделах данного раздела приводятся общие сведения о драйвере JDBC, включая системные требования, необходимые для его использования, а также описание его использования и ссылки для получения дополнительных сведений.
