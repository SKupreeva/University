Интерфейс ISQLServerDataSource

Фабрика для создания соединений с источником данных, представленным данным объектом. Этот интерфейс добавлен в версии 3.0 драйвера SQL Server JDBC.

Пакет: com.microsoft.sqlserver.jdbc

Расширяет: java.sql.CommonDataSource

Синтаксис

public interface ISQLServerDataSource

Замечания

Этот интерфейс реализуется классом Класс SQLServerDataSource Class.

Этот интерфейс обеспечивает доступ к следующим свойствам, определяемым Драйвер Microsoft JDBC для SQL Server.
