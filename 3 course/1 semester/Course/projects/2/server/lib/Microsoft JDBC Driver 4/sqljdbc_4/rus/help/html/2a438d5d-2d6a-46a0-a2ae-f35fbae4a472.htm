Поля

Имя

Описание

CONCUR_SS_OPTIMISTIC_CC

Указывает тип оптимистичного параллелизма чтения-записи в SQL Server без блокировки строк.

CONCUR_SS_OPTIMISTIC_CCVAL

Указывает тип оптимистичного параллелизма чтения-записи в SQL Server без блокировки строк.

CONCUR_SS_SCROLL_LOCKS

Указывает тип оптимистичного параллелизма чтения-записи в SQL Server с блокировкой строк.

TYPE_SS_DIRECT_FORWARD_ONLY

Используется для указания быстрых однопроходных курсоров только для чтения SQL Server.

TYPE_SS_SCROLL_DYNAMIC

Используется для указания типа динамичных курсоров SQL Server.

TYPE_SS_SCROLL_KEYSET

Используется для указания типа курсоров, управляемых набором ключей SQL Server.

TYPE_SS_SCROLL_STATIC

Используется для указания типа статических курсоров SQL Server.

TYPE_SS_SERVER_CURSOR_FORWARD_ONLY

Используется для указания быстрых однопроходных курсоров только для чтения SQL Server.
