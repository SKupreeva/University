Метод executeUpdate (java.lang.String, int)

Выполняет заданную инструкцию SQL и с помощью определенного флага уведомляет драйвер Драйвер Microsoft JDBC для SQL Server о том, что автоматически сформированные ключи, созданные этим объектом SQLServerStatement, должны быть доступны для получения.

Синтаксис

public final int executeUpdate(java.lang.String sql,
                               int flag)

Параметры

sql

Значение типа String, содержащее инструкцию SQL.

flag

Значение типа int, которое указывает, нужно ли обеспечить доступ к автоматически формируемым ключам. Это должна быть одна из следующих констант:

RETURN_GENERATED_KEYS

NO_GENERATED_KEYS

Возвращаемое значение

Значение типа int, которое указывает число обработанных строк, либо значение 0 для инструкций языка определения данных.

Исключения

SQLServerException

Замечания

Этот метод executeUpdate указывается методом executeUpdate в интерфейсе java.sql.Statement.

Если при выполнении хранимой процедуры значение счетчика обновлений стало больше единицы либо было создано более одного результирующего набора, пользуйтесь для выполнения хранимой процедуры методом execute.
