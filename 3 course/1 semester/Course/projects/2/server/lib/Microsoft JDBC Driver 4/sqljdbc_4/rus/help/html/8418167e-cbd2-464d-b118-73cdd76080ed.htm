Метод executeBatch (SQLServerPreparedStatement)

Отправляет пакет команд базе данных для выполнения. В случае успешного выполнения всех команд возвращает массив из количества операций обновления, выполненных той или иной командой.

Синтаксис

public int[] executeBatch()

Возвращаемое значение

Массив значений типа int, содержащий счетчики обновлений.

Исключения

SQLServerException

java.sql.BatchUpdateException

Замечания

Этот метод executeBatch определяется методом executeBatch в интерфейсе java.sql.Statement.

Драйвер SQL Server JDBC Driver 3.0 соответствует рекомендации JDBC 4.0, согласно которой вызов метода CallableStatement.executeBatch (наследуемого от PreparedStatement) должен вызывать исключение BatchUpdateException, если хранимая процедура принимает параметры OUT или INOUT либо возвращает что-либо, кроме счетчика обновления.

Этот метод переопределяет SQLServerStatement.executeBatch.
