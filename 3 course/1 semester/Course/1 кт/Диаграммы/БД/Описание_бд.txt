Таблица Employee

Имеет поля E_ID - код клиента, подключающегося к бд
	   Login - логин клиента
	   Passw - пароль клиента
По значениям полей Login и Password при входе в систему определяется должность работника и уровень его доступа к хранимым в бд данным
(Директор и Аналитик).


Таблица Department 

Имеет поля D_ID - код отдела
	   DepName - имя пользователя
Данная таблица отвечает за отдел, к которому принадлежат работники предприятия.


Таблица Calculation

Имеет поля C_ID - код расчета
	   x1 - значение переменной x1
	   x2 - значение переменной x2
	   x3 - значение переменной x3
	   x4 - значение переменной x4
	   x5 - значение переменной x5
	   x6 - значение переменной x6
	   x7 - значение переменной x7
	   x8 - значение переменной x8
	   x9 - значение переменной x9
	   AverageValue - среднее значение финансовых показателей
	   ResultValue - результат расчета
Данная таблица отвечает за расчет прогноза платежеспособности предприятия на основе модели банкротства Фулмера. Таблица содержит поля для
сохранения как исходных, так и расчетных данных.   


Таблица Manager 

Имеет поля М_ID - персональный код Директора
	   D_ID_FK - код отдела, к которому относится данный пользователь(внешний ключ)
	   E_ID_FK - код клиента(внешний ключ)
	   Name - имя пользователя
	   Surname - фамилия пользователя
Данная таблица отвечает за персональную информацию о Директоре.


Таблица Analyst 

Имеет поля A_ID - персональный код Аналитика
	   D_ID_FK - код отдела, к которому относится данный пользователь(внешний ключ)
	   E_ID_FK - код клиента(внешний ключ)
	   Name - имя пользователя
	   Surname - фамилия пользователя
Данная таблица отвечает за персональную информацию об Аналитике.


Таблица Report 

Имеет поля R_ID - код отчета
	   A_ID_FK - персональный код аналитика, который составил отчет(внешний ключ)
	   C_ID_FK - код расчета, проводимого для составления данного отчета(внешний ключ)
	   RepDate - дата составления отчета
Данная таблица отвечает за информацию об отчете, предоставляемом директору по окончанию расчета прогноза.
