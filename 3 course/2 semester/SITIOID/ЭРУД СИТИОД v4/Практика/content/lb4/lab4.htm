Лабораторная работа №4 Построение отчетности с возможностью фильтрации и навигации


Задание.

  1. Создать в MySQL базу данных из одной таблицы (6 полей, 50 строк).

  2. Используя SpagoBIStudio создать 2 отчета (исходных данных и на выборку данных), структура которых приведена ниже.

  3. Используя SpagoBI-Server, отобразить созданные отчеты.
